# **Getting Started**

### 1. Define Database

Define database user and database password in **.env** file.

### 2. Installing Packages

```bash
npm i
```

### 3. Migrate Database

```bash
node ace migration:run
```

### 4. Seed Database

```bash
node ace db:seed
```

### 5. Run Server
```bash
npm run dev
```
